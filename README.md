# BetterARAM
 **League of Legends ARAM Enhancement Tool**
 
 **칼바람 유틸**

 ## features
  - gamechat commands
  - auto bench swap
  - win/lose streak messages
  - load runes
  - auto accept queue
  - simple and easy-to-use options

 ## 기능
  - 채팅 명령어
  - 벤치 자동 스왑
  - 연승/연패 메시지
  - 룬 불러오기
  - 대기열 자동 수락
